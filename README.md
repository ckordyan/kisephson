# Kisephson

A personal website and dashboard for favorite links, useful calculators, recipes, notes, and car research.

## Local Preview

Open `index.html` directly, or run a small local server:

```bash
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## GitHub Pages

For a personal GitHub Pages site, create a repository named:

```text
Kisephson.github.io
```

Push this project to that repository and enable GitHub Pages from the repository settings.
